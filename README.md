# SQL Orders Analytics

A MySQL Workbench project demonstrating table creation, sample data insertion, SELECT queries, CASE WHEN logic, NULL handling, filtering, and revenue calculations using SQL.

## 📋 Overview

This project provides a comprehensive SQL learning resource with practical examples of database design and querying. It demonstrates core SQL concepts through a realistic orders and analytics scenario, perfect for those learning SQL fundamentals or practicing advanced query techniques.

## ✨ Features

- **Table Creation**: Structured schema design for orders management
- **Sample Data**: Pre-populated datasets for testing and learning
- **SELECT Queries**: Comprehensive query examples for data retrieval
- **CASE WHEN Logic**: Conditional logic for data transformation
- **NULL Handling**: Best practices for handling NULL values
- **Filtering**: Advanced filtering techniques with WHERE clauses
- **Revenue Calculations**: Real-world examples of aggregate functions and calculations

## 🚀 Getting Started

### Prerequisites

- MySQL Server (5.7 or higher)
- MySQL Workbench or any MySQL client

### Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/ReetiShah/sql-orders-analytics.git
   cd sql-orders-analytics
   ```

2. Open the MySQL Workbench project file or import the SQL scripts

3. Execute the SQL scripts in the following order:
   - Table creation scripts
   - Sample data insertion scripts
   - Query examples

## 📚 Project Structure

This project includes:
